Environnements
---------------------------------
Java 8
Maven 3.6.3


Modules utilisés
---------------------------------
Springboot
Junit 5
Lombok
Swagger
H2 database


Récupération des sources
---------------------------------
git clone https://github.com/julio0701/alten-test-booking-api.git


Lancement de l'application
---------------------------------
1)	Se rendre dans le dossier du projet cloné et y ouvrir un terminal
2)  taper la commande: mvn spring-boot:run
3)  accéder à l'adresse: http://localhost:8087/swagger-ui.html
